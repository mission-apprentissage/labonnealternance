# Présentation du service

L'objectif de ce service est de permettre à ses utilisateurs de trouver plus facilement et plus rapidement une formation et un emploi en alternance. Pour cela, il offre d'une parte une information complète et centralisée sur la marché, et accompagne d'autre part les mises en relations entre candidats, entreprises et centres de formation.

Le site internet La bonne alternance, en cours d'amélioration, est disponible ici : [https://labonnealternance.apprentissage.beta.gouv.fr/](https://labonnealternance.apprentissage.beta.gouv.fr/)

Afin de faciliter les démarches de recherche des candidats à l'alternance, le service propose l'exposition de ses données en marque blanche via API ou widget. L'objectif étant de simplifier les démarches des candidats et  leur délivrant l'information où ils se trouvent déjà, plutôt que de les contraindre à utiliser un site en particulier.

Dans la suite de cette documentation, vous trouverez ainsi la documentation nécessaire à l'exploitation des widgets et/ou API La bonne alternance.
